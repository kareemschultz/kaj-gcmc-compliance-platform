Tracks official and legal documents linked to each Customer.

## Core Fields

| Field         | Type    | Description                                      |
|---------------|---------|--------------------------------------------------|
| document_type | Select  | Document category / type                         |
| issue_date    | Date    | Date issued                                      |
| expiry_date   | Date    | Valid until (if applicable)                      |
| issuer        | Select  | GRA / NIS / DCRA / Government / Court / Other   |
| attached_file | Attach  | PDF or scan                                      |
| remarks       | Small Text | Additional notes                             |
